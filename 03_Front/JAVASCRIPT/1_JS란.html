자바스크립트란?

웹브라우저에서 사용되는 인터프리터방식의 스크립트 언어 브라우저에 js엔진이 내장되어있어(크롬의 v8엔진, 사파리의 자바스크립트코어, 파이어폭스의 스파이더몽키 등) 브라우저에서 설치없이 실행 가능

  1. inline : 태그 내부에 코드를 직접 작성하는 방식
  2. internal : head태그 내부에 script태그를 이용해 작성하는 방식
  3. external : 외부 js파일을 만들어 script태그를 이용해 불러오는 방식

1) inline

태그 on이벤트명="js코드"



2) internal

head태그 내부에 script태그를 이용해 작성하는 방식

3) external

외부 js파일을 만들어 script태그를 이용해 불러오는 방식
