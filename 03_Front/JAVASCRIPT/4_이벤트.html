Event

이벤트 : 브라우저에서 발생하는 모든 동작(click, keyup, keydown, mouseover, mouseout, load, drag) 이벤트 핸들러 : 트정 이벤트가 발생했을 때 실행되는 함수

1) 고전 이벤트 모델 - element.onclick = 함수

동일 이벤트에 1개만 등록됨

2) 표준 이벤트 모델 - element.addEventListener(이벤트명, 함수)

동일 이벤트에 여러 핸들러 등록 가능, removeEventListener()로 제거 가능

3) 이벤트 객체

preventDefault() - 기본동작막기
